# 404 Page — Design Document

**Date:** 2026-02-27
**Status:** Approved

## Overview

Add a custom 404 "Page Not Found" to ResumeForge that matches the existing dark design system and provides clear navigation recovery.

## Routing

Next.js App Router automatically serves `app/not-found.tsx` for any unmatched URL. No additional route config is needed.

The file lives at `app/not-found.tsx` (root level, outside the `(main)` route group). To include Header and Footer, it imports them directly from `@/components/`.

## Layout

Vertically and horizontally centered content within a `min-h-screen` wrapper using the base background (`#0A0A0B`).

### Content block (centered column)

| Element | Style |
|---------|-------|
| "404" | `font-serif`, ~8rem, `#FF5C00` accent orange |
| "Page not found" | H1, white, ~2rem |
| Body copy | "The page you're looking for doesn't exist or has been moved." `text-secondary` (`#8B8B90`) |
| Primary CTA | "Go Home" — orange button (`#FF5C00` → `#e05200` hover) |
| Secondary CTA | "Browse Templates" — ghost/outline button |

### Animation

Framer Motion `fadeUp` on the content block — matches the landing page convention (`opacity 0→1`, `y 24→0`). Client Component (`"use client"`) required.
